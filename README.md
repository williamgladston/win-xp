📌 Overview

WinXP Web OS Clone is a front-end project that simulates the look and feel of the original Windows XP operating system directly in the browser. It replicates core UI behaviors such as draggable windows, a start menu, desktop icons, and basic applications.

This project is inspired by popular web-based OS recreations like winXP, which demonstrate how nostalgic desktop environments can be rebuilt using modern JavaScript frameworks.  

⸻

✨ Features

* 🪟 Draggable, resizable, minimizable windows
* 🧭 Functional Start Menu & Taskbar
* 🖥️ Desktop icons with app launching
* 📂 File-system-like navigation (UI-based)
* 🎨 Windows XP styled UI (classic theme)
* ⚡ Smooth animations and transitions
* 🌐 Fully runs in the browser (no backend required)
